문화 이론에서 배운 자바/이클립스

자바라는 언어에 대해서 배우며 이클립스 프로그램 사용법도 익혔다. 자바의 특징과 다른 언어들과의 차별점이 무엇인지 배우고 간단한 계산기와 모듈화에 대해서 배웠다.

응용 프로그래밍 개발

자바스크립트라는 언어를 새롭게 배웠는데, 조금 어려웠지만 재미있었다. HTML로 사이트의 구성을 만들고 JavaScript로 실제 웹페이지의 동작을 만드는 부분이 꽤 재미있었다. HTML, CSS, JS를 모두 사용하여 나만의 홈페이지를 만들었다.

코드업 기초 100제를 풀어보면서 C언어의 기초를 다질 수 있었다. 또한 Java로도 똑같은 문제를 풀어보면서 프로그래밍 언어의 차이도 익혔다

PHP 언어를 이용해서 SQL 쿼리 문을 실행해보고, 결과 값을 HTML로 출력해보았는데, 실제 서버와 브라우저의 통신처럼 되는게 재밌었다

ㅇ

무료 호스팅인 닷홈을 이용해서 웹 서버를 호스팅했는데, 그 과정에서 사용한 파일질라 프로그램이다. 조금은 복잡했지만 호스팅만 되어있다면 언제 어디서든 내가 만든 서버에 접근할 수 있다는게 재밌었다.

PHPMyAdmin 서버를 직접 셋업도 해보고, 닷홈을 이용해서 열기도 해봤다. SQL 쿼리문을 배우고 데이터를 직접 넣고 수정하는게 나중에 도움이 될 것 같았다.

스마트문화 앱 콘텐츠 제작

자바스크립트라는 언어를 새롭게 배웠는데, 조금 어려웠지만 재미있었다. HTML로 사이트의 구성을 만들고 JavaScript로 실제 웹페이지의 동작을 만드는 부분이 꽤 재미있었다. HTML, CSS, JS를 모두 사용하여 나만의 홈페이지를 만들었다.

Java의 Swing을 이용해서 기본적인 창을 띄워보았다. 아무 것도 없는 기초적인 창이지만 이 창 하나를 띄우기 위해서 필요했던 코드를 생각하면 가슴이 아프다.

직접 배경화면을 구한 뒤에 포토샵을 이용해서 로고 디자인을 해보았다. 추가로 이미지를 자바 GUI창에 넣어 프로그램 실행 시 이미지가 보이도록 해보았다. 생각보다 이미지를 잘 만든 것 같아 뿌듯했다.

자바의 자료형에 대해서 배웠고, 2진수와 10진수가 어떤 것인지, 어떻게 서로 변환하는지 방법에 대해 배웠다. 프로그래밍이 아니라 약간 컴퓨터에 대해서 더 자세히 배우는 느낌이었다.

기존에 C언어로 해보았던 코드업 기초 100제를 자바로 다시 풀어보았다. 똑같은 문제지만 코드가 완전히 달라서 신기했다. 자바의 기초적인 IO를 배우는 과정이 재미있었다.
